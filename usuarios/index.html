    Usuarios

    ARCHIVOS DE CONFIGURACIÓN

    Los sistemas GNU/Linux gestionan los usuarios mediante archivos de configuración:

    • Fichero /etc/passwd → Almacena información de los usuarios.
    • Fichero /ect/group → Almacena información de los grupos del sistema
    • Fichero /etc/shadow → Almacena información de las contraseñas de los usuarios.
    • Fichero /etc/gshadow → Almacena información de las contraseñas de los grupos.
    • Fichero /etc/adduser.conf → Parámetros por defecto para el comando adduser
    • Fichero /etc/login.defs → Configuración por defecto: mascara, ids, caducidad contraseña.
    • Fichero /etc/skel → Esqueleto con carpetas y archivos al crear un usuario.

    USUARIOS

    Crear

    La estructura básica del comando es : 

        useradd agustin -m -d /home/profesor -s /bin/bash

    Los principales parámetros a utilizar:

    Parámetro Descripción
    -c Descripción del usuario.
    -d Directorio home (no lo crea, debe existir)
    -e Fecha de caducidad de la cuenta
    -g Grupo principal (no lo crea, debe existir)
    -G Establece los grupos secundarios a los que pertenecerá el usuario.
    -s Shell por defecto (/bin/bash, /bin/sh, etc)
    -u Identificador del usuario (ID, no debe existir previamente)
    -m Crea el directorio home asignado con la opción –d
    -f Días de gracia desde que caduca contraseña hasta que deshabilita la cuenta.

    Editar

    La estructura básica del comando es : 

        usermod agustin -s /bin/bash

    Los principales parámetros a utilizar:

    Parámetro Descripción
    -d Cambia el directorio personal
    -m junto con –d mueve los ficheros del directorio personal al nuevo, y si no existe el
    nuevo directorio, lo crea
    -e Cambia la fecha de expiración de la cuenta
    -g Permite cambiar el grupo principal del usuario
    -G Cambia la lista de grupos a los que pertenece el usuario
    -a Junto con –G añade grupos al usuario
    -l Modifica el login de usuario. El nombre de la carpeta personal también cambia.
    -L Bloquear cuenta. No podrá iniciar sesión hasta desbloquearda
    -U Desbloquear cuenta de usuario
    -s Cambiar la shell por defecto

    Borrar

    La estructura básica del comando es : 

        userdel -r agustin

    GESTIONAR GRUPOS

    Crear

    La estructura básica del comando es : 

        groupadd profesores -g 3001

    Editar

    La estructura básica del comando es : 

        groupmod profesores -g 3001

    Borrar

    La estructura básica del comando es : 

        groupdel profesores

    USUARIOS vs GRUPOS

    Añadir

    Añadir usuarios a un grupo local, como grupo principal

        usermod -g grupo usuario

    Como grupo secundario:

        usermod -G grupo usuario  // Opción 1
    gpasswd -a usuario grupo  // Opción 2

    Para eliminar usuarios de un grupo

        gpasswd –d usuario grupo

    OTROS COMANDOS

    Otros comandos relacionados con la gestión de usuarios:

    • groups -> Muestra los grupos a los que pertenece un usuario
    • chfn -> Permite a un usuario cambiar su propio campo descripción.
    • chsh -> Permite a un usuario cambiar su shell por defecto
    • chpasswd -> Cambiar la contraseña, de forma no interactiva:
        echo "agustin:1234567" | sudo chpasswd
